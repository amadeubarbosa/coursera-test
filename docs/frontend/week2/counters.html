Lessons about CSS counters

Ordinary simplest list possible renders to:

  1. item one
  2. item two
  3. item three
  4. item four

Rendering a sublist

We can render a sublist in a pretty way using only CSS2 counters, like that:

  1. one
  2. two
    1. two.one
    2. two.two
    3. two.three
    4. two.four
  3. three
    1. three.one
    2. three.two
      1. three.two.one
      2. three.two.two
    3. three.three
      1. three.three.one
      2. three.three.two
      3. three.three.three
      4. three.three.four
  4. four
